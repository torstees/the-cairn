{# Shared alias-name renderers for tune listings. alias_summary(aliases): "Also known as: ..." — used where a tune's row already has room for a second line (tunes/, boxes/{id}, lists/{id}, progress/). Renders inline; callers wrap the call in whatever block-level spacing fits their own layout. alias_tooltip(names, align): the hover tooltip box itself (a plain list of name strings, not TuneAlias objects) — exposed directly for layouts with no room for alias_summary's visible text, which instead make an existing element (e.g. the tune title) the hover trigger and drop this in as a child. `align` controls which edge the tooltip is anchored to ("left", the default, or "right") — match it to where the trigger sits in the row, since a left-anchored tooltip on a trigger near the row's right edge routinely overflows the viewport (sets/{id}'s member list hit exactly this before the trigger there moved onto the title). The trigger element needs `relative` positioning for this to anchor correctly. Usage: import alias_summary and alias_tooltip from this file. Call alias_summary(tune.aliases) directly, or for a custom trigger (e.g. sets/detail.html's member title), add `relative` to the trigger element and drop alias_tooltip(names) inside it. #} {% macro alias_tooltip(names, align="left") %} {% if names %} {# No margin between the trigger and this box (top-full, no mt-*): a gap would be dead space outside both the trigger's box and this element, losing :hover the instant the cursor crosses it on the way down to read the tooltip, which un-hides whatever is really beneath that pixel — a row-level ABC hover-preview trigger, once those existed (#135). Visual spacing instead comes from this box's own border/shadow/padding. #} {{ names | join(', ') }} {% endif %} {% endmacro %} {% macro alias_summary(aliases, max_shown=5) %} {% if aliases %} {% set names = aliases | map(attribute='name') | list %} {# data-abc-preview-id="" shadows a row-level ABC hover-preview trigger (see static/js/app.js's closest("[data-abc-preview-id]") delegation): hovering this area shows only the alias tooltip, not the ABC preview. #} Also known as: {{ names[:max_shown] | join(', ') }}{% if names | length > max_shown %}…{% endif %} {% if names | length > max_shown %}{{ alias_tooltip(names) }}{% endif %} {% endif %} {% endmacro %}
